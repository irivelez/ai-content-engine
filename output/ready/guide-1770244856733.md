## Qué es Claude Code (y por qué te importa)

Piénsalo así: Claude en el chat web es como hablar con un asistente por teléfono. Le pides cosas y te responde.

Claude Code es como tener a ese asistente **sentado frente a tu computadora**. Puede leer tus archivos, crear documentos, organizar carpetas, analizar datos y hasta construir herramientas para ti.

La diferencia clave: no solo te *dice* qué hacer. Lo *hace*.

¿Tienes 50 archivos PDF que necesitas organizar? Claude Code los lee y los renombra.
¿Necesitas convertir una hoja de Excel en un reporte bonito? Claude Code lo genera.
¿Quieres una página web simple para tu negocio? Claude Code la construye.

Sin que tú escribas código. Tú le dices qué quieres en español, y él se encarga.

## Cómo instalar Claude Code (paso a paso)

Esto toma menos de 10 minutos. Te lo prometo.

**Paso 1: Instala Node.js**

Ve a → [nodejs.org](https://nodejs.org)
Descarga la versión LTS (el botón verde grande).
Instálalo como cualquier programa: siguiente, siguiente, finalizar.

**Paso 2: Abre la terminal**

- En Mac → busca "Terminal" en Spotlight (Cmd + Espacio)
- En Windows → busca "PowerShell" en el menú de inicio

No te asustes. La terminal es solo una ventana donde escribes comandos. Nada más.

**Paso 3: Instala Claude Code**

Copia y pega esto en la terminal:

`npm install -g @anthropic-ai/claude-code`

Presiona Enter. Espera a que termine.

**Paso 4: Conéctalo a tu cuenta**

Escribe `claude` en la terminal y presiona Enter.
Te va a pedir que inicies sesión con tu cuenta de Anthropic. Sigue las instrucciones en pantalla.

Listo. Ya tienes Claude Code funcionando. ⚡

## 5 cosas que puedes hacer HOY sin saber programar

Aquí es donde se pone bueno. Estos son casos de uso reales que yo misma he probado:

**1. Analizar documentos en segundos**

Navega a la carpeta donde tienes tus archivos (escribe `cd` seguido de la ruta) y dile:

→ *"Lee todos los PDFs en esta carpeta y hazme un resumen ejecutivo de cada uno en un solo documento"*

Funciona con contratos, reportes, propuestas. Cualquier documento.

**2. Organizar archivos como un profesional**

→ *"Revisa todos los archivos en esta carpeta. Crea subcarpetas por tema y mueve cada archivo donde corresponda. Muéstrame qué hiciste antes de confirmar."*

Esa última frase es clave: "muéstrame antes de confirmar". Claude Code te pide permiso antes de ejecutar cambios. Tú siempre tienes el control.

**3. Crear presentaciones y documentos**

→ *"Crea una presentación en HTML sobre tendencias de IA en LATAM para 2025. Que tenga 10 slides, diseño profesional, y datos específicos."*

Te genera un archivo que abres en el navegador y se ve como si la hubiera hecho un diseñador.

**4. Transformar datos de Excel**

→ *"Lee el archivo ventas-2024.xlsx. Dame un análisis con los 5 productos más vendidos, la tendencia mensual, y genera gráficas. Pon todo en un reporte HTML."*

Lo que antes tomaba horas en Excel, ahora toma minutos.

**5. Construir herramientas simples**

→ *"Crea una calculadora web donde yo pueda ingresar mis gastos mensuales y me muestre un resumen con gráficas de dónde se va mi dinero."*

Claude Code la construye completa. Tú la abres en el navegador y funciona.

## Los 3 secretos para que funcione bien

Después de meses de uso diario, esto es lo que he aprendido:

**Sé específica con lo que quieres.** No digas "ayúdame con mis archivos". Di "lee los 20 PDFs en la carpeta Contratos, extrae el monto y la fecha de cada uno, y ponlos en una tabla ordenada por fecha".

**Pide confirmación antes de cambios.** Agrega siempre: "muéstrame el plan antes de ejecutar" o "pide confirmación antes de modificar archivos". Es tu red de seguridad.

**Trabaja en una carpeta organizada.** Antes de empezar, crea una carpeta para tu proyecto. Navega ahí con `cd nombre-de-carpeta`. Claude Code trabaja mejor cuando sabe exactamente dónde está.
