## Qué es un "agente" (en 30 segundos)

Un agente de IA no es un chatbot. Un chatbot responde preguntas. Un agente **ejecuta tareas completas.**

Le dices: "Analiza las ventas del Q1, identifica los 3 productos con mayor caída, y arma un resumen ejecutivo con recomendaciones."

Y lo hace. Paso por paso. Solo. Pidiéndote confirmación cuando la necesita.

Tú no escribes código. Tú **diriges.**

## 5 pasos para empezar con ingeniería agéntica HOY

### 1. Identifica una tarea repetitiva que te roba más de 2 horas/semana

No empieces con algo complejo. Busca algo aburrido y repetitivo:

- Consolidar datos de varias fuentes en un reporte
- Redactar correos de seguimiento a clientes
- Resumir documentos largos para tu equipo
- Preparar agendas de reuniones basadas en pendientes

**Escríbela.** Esa es tu primera misión agéntica.

### 2. Descompón la tarea en pasos claros

Aquí está el skill real. Los agentes no leen mentes — necesitan instrucciones estructuradas.

Ejemplo: "Hacer el reporte semanal" se convierte en:

→ Paso 1: Revisar los datos de ventas en [fuente]
→ Paso 2: Comparar contra la semana anterior
→ Paso 3: Identificar las 3 variaciones más grandes
→ Paso 4: Escribir un resumen de 5 bullets para el director
→ Paso 5: Formatear como correo listo para enviar

**Esa descomposición ES la ingeniería agéntica.** Si puedes describir los pasos, puedes dirigir un agente.

### 3. Usa este prompt framework para dirigir al agente

Copia y adapta esto en Claude, ChatGPT o el modelo que uses:

*"Actúa como [ROL]. Tu tarea es [OBJETIVO ESPECÍFICO]. Sigue estos pasos en orden: [LISTA DE PASOS]. El formato de salida debe ser [FORMATO]. Antes de entregar el resultado final, verifica que [CRITERIO DE CALIDAD]."*

Ejemplo real:

*"Actúa como analista de negocio senior. Tu tarea es crear un resumen ejecutivo de las ventas del mes. Sigue estos pasos: 1) Analiza los datos que te voy a compartir, 2) Identifica los 3 productos con mejor desempeño y los 3 con peor, 3) Calcula la variación porcentual vs. mes anterior, 4) Escribe 5 hallazgos clave en bullets. El formato debe ser un correo profesional de máximo 200 palabras dirigido al director comercial. Antes de entregar, verifica que todos los números sean consistentes."*

### 4. Itera como jefe, no como usuario

Aquí es donde el 90% de la gente falla. Le dan UNA instrucción al agente, reciben algo mediocre, y dicen "la IA no sirve."

Los buenos ingenieros agénticos hacen esto:

- **Revisan** el primer resultado (no lo aceptan ciego)
- **Dan feedback específico:** "El tono es muy formal, hazlo más directo" o "Falta el dato de margen por producto"
- **Refinan** el prompt original para la próxima vez
- **Guardan** los prompts que funcionan (tu biblioteca personal)

Es exactamente como dirigir a un equipo: das instrucciones claras, revisas entregables, ajustas.

### 5. Encadena agentes para flujos completos

El nivel avanzado: usar un agente para alimentar a otro.

→ Agente 1: Extrae datos clave de 5 documentos PDF
→ Agente 2: Compara los datos y encuentra patrones
→ Agente 3: Escribe la presentación final con los hallazgos

Esto ya es **orquestación real.** No necesitas herramientas especiales para empezar — puedes hacerlo manualmente, copiando el output de una conversación como input de otra.

Cuando estés lista para automatizar, herramientas como Zapier AI, Make o n8n te permiten conectar estos pasos sin código.
